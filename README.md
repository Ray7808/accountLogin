# 實作登入功能

此專案主要透過 Express 和 handlebars 實作使用者輸入正確信箱跟密碼時，回傳歡迎使用者回來的頁面。

## 產品功能

-   透過 Express 以及模板引擎 handlebars，產生登入跟登入成功的頁面

## 安裝流程

1. git clone 下載檔案

    ```
    git clone https://github.com/Ray7808/shortLink.git
    ```

2. 安裝相關套件

    ```
    npm install
    ```

3. 執行程式
    ```
    npm run start
    ```

## 使用的套件

-   Bootstrap
-   Express
-   Handlebars
-   Nodemon
-   body-parser
